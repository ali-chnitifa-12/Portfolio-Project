# Space-Themed Developer Portfolio

A professional, high-performance developer portfolio featuring futuristic design, 3D animations, and a seamless user experience. Built with Next.js, GSAP, and Tailwind CSS.



## 🚀 Features

- **Futuristic 3D Carousel**: Interactive project showcase using GSAP and 3D transforms.
- **Dynamic Starfield Background**: Immersive space-themed visual experience.
- **Responsive Design**: Fully optimized for all screen sizes with a mobile-first approach.
- **Glassmorphism UI**: Modern, sleek components with high-end transparency effects.
- **Interactive Contact Form**: Integrated with **Resend** for real-time email notifications.
- **Smooth Page Transitions**: Cinematic entrance and scroll-triggered animations.
- **Dynamic Projects**: Easy-to-manage project data with custom screenshots.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/)
- **Email Service**: [Resend](https://resend.com/)
- **Language**: TypeScript

## 📂 Project Structure

- `src/components/sections`: Contains the main sections of the website (Hero, About, Projects, Contact, etc.).
- `src/components/StarfieldBackground.tsx`: Custom canvas-based space animation.
- `src/app/api/contact/route.ts`: API route for sending emails via Resend.
- `public/projects`: High-quality screenshots for the 3D portfolio carousel.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ali-chnitifa-12/Portfolio-Project.git
   cd Portfolio-Project
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up Environment Variables**:
   Create a `.env.local` file in the root directory and add your Resend API Key:
   ```env
   RESEND_API_KEY=your_resend_api_key
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```
